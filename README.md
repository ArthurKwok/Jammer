# Jammer

## Requirements
### Python packages
* music21
* pysox
### linux packages
* fluidsynth
* SoX
* MMA 20.02
### others
* OrpheusGM Soundfont: https://musical-artifacts.com/artifacts/1213
  
## Installation
1. copy lib/jammer into mma's library
2. in shell type `mma -G`
3. put OrpheusGM soundfont in ./downloads

## Run
in any CLI type `python ./src/producer.py`
